## Architecture
This project serves as a monorepo for the Divya Bihar platform.

*   **Frontend**: Next.js (Visual/Interactive Layer) - Located in `/frontend`
*   **Backend**: NestJS (API/Business Logic Layer) - Located in `/backend` (To be initialized)
*   **Database**: PostgreSQL
*   **Infrastructure**: Modular Monolith / Microservices ready

## Core Features
1.  **Destinations Module**: Guided info on temples, history, and maps.
2.  **Trip Planner**: ITinerary builder and route planning.
3.  **Hotels & Services**: Booking and listings.
4.  **Blog/CMS**: SEO-friendly content.
